# Web Search

Your agent has web search built in via Brave Search. No setup needed.

## How It Works

Your agent can search the web as part of answering your questions. Just ask naturally:
- "What's the latest news about X?"
- "Find me the best Python library for Y"
- "Look up how to do Z"

## Tips

- Web search uses tokens from your monthly allowance, so it costs a bit more per message
- For research tasks, ask your agent to search once and summarize rather than searching repeatedly
- Your agent can also fetch and read specific URLs if you give it a link
