⚠️ Common Issues This Week

If you hit one of these, you're not alone:

🔧 Discord bot won't reply

Still the #1 issue. Message Content Intent must be enabled in the Discord Developer Portal. We wrote a full tutorial to fix this once and for all.

🔧 502 errors on agent subdomain

Several users reported intermittent 502 errors. If your agent is down, post in #help with your signup email and the team can check server-side.

🔧 "My agent forgot everything"

This keeps coming up. Agents don't automatically remember between sessions — they need to write to files. Read How Memory Works to understand why and how to fix it.

🔧 Can't paste images in web chat

The web UI is text-only for now. Use Discord or Telegram to send files/images to your agent. See Working with Files.
