Working with Files

Sending photos, documents & more to your heyron agent — and getting files back.

⏱ 3 minute read

One of the most common questions we get: "How do I send my agent a screenshot?" or "I tried pasting an image in the web chat and nothing happened."

You're not doing anything wrong. File attachments depend on which platform you're using to talk to your agent. Here's the full picture.

Where File Attachments Work

💬
✅ Full Support

Discord

Paste, drag-and-drop, or use the attachment button. Images, documents, PDFs, code files — it all works. Your agent can see and analyze everything you send.

✈️
✅ Full Support

Telegram

Send photos, documents, and voice messages directly — just like you would to any contact. Your agent receives and processes them automatically.

🌐
❌ Text Only

Web Dashboard (heyron.ai/chat)

The web chat is text only right now. The paperclip button, copy-paste, and drag-and-drop won't work for files. This is a known limitation and we're actively working on it.

💡 Tip
If you need to send your agent an image or file, use Discord or Telegram. You can also use agentmail.to to email files to your agent — it's super simple to set up. These are the best ways to share files with your agent right now.

How to Share Files with Your Agent

💬 Through Discord

Just paste or upload files directly in your agent's DM or channel. Works exactly like sending files to any Discord user.

  • Paste screenshots with Ctrl+V / Cmd+V
  • Drag and drop files into the chat
  • Click the + button to browse and attach

✈️ Through Telegram

Send photos, documents, or voice messages as you normally would. Your agent picks them up instantly.

  • Use the 📎 attachment button for documents
  • Send photos directly from your camera or gallery
  • Voice messages are transcribed automatically

🌐 Through the Web Dashboard

Since file uploads aren't supported yet, here's what you can do:

  • Describe what you need in text — your agent is great at working from descriptions
  • Paste text content directly (code, logs, error messages)
  • For images: upload to Imgur or Google Drive, then share the link
⚡ Workaround
Need your agent to see an image but you're on the web chat? Upload it to any public image host and paste the URL. Your agent can fetch and analyze images from links.

Your Agent Can Create Files For You

While getting files to your agent depends on the platform, your agent can create files on its own — regardless of how you're chatting.

📝 What your agent can create

  • Code files in any language
  • Documents and notes
  • HTML pages and web content
  • CSV and structured data
  • Configuration files, scripts, and more

Your agent saves files to its workspace — a dedicated directory on its host machine. To get those files back to you:

📤 Getting files from your agent

  • Ask it to share in chat — your agent can paste file contents directly into the conversation. Great for text, code, and short documents
  • GitHub integration — if connected, your agent can commit and push files to a repo you can access
  • Publish it — for HTML pages or websites, your agent can deploy to here.now or GitHub Pages and give you a live URL
  • Email it to you — with agentmail.to set up, your agent can email files directly to you
  • Share the file path — your agent can tell you where it saved a file in its workspace, and you can browse files in the Agents tab of your dashboard

Quick Troubleshooting Checklist

🛠️ If file sending isn't working, check these first

  • You're in the right channel: Test in the same Discord DM/channel where your agent already replies to text.
  • Discord Message Content Intent is enabled: If this is off, your bot may miss file + message context.
  • Bot has permissions: In server channels, confirm the bot can View Channel, Send Messages, and Read Message History.
  • File size is reasonable: Extremely large uploads may fail before your agent sees them.
  • Try a known-good test: Upload a small PNG screenshot with a simple prompt like "What text is in this image?".
Still stuck?
If your agent answers normal text messages but ignores file uploads, reconnect Discord integration and re-check intents/permissions. Most upload issues are configuration, not model problems.

Common Questions

"Why can't I paste images in the web chat?"

The web dashboard at heyron.ai/chat is text-only for now. It doesn't support image paste, file uploads, or drag-and-drop. This isn't a bug — it's a feature that hasn't shipped yet.

In the meantime, use Discord or Telegram for anything involving files or images. Both have full support.

"Can my agent see images I send?"

Yes! When you send an image via Discord or Telegram, your agent can analyze it — describe what's in it, read text from screenshots, identify objects, review designs, and more.

This uses vision capabilities built into the AI model. Just send the image and ask your agent what you need.

"Can my agent create images?"

Your agent can create HTML pages, SVG graphics, and code-based visuals natively. For AI-generated images (like DALL·E), it depends on your setup — some users connect image generation tools as skills.

Your agent can also write detailed image prompts that you can use in Midjourney, DALL·E, or other generators.

"How do I get files my agent created?"

A few options:

• Ask it to display the content — your agent can paste file contents right into the chat.
• Push to GitHub — if your agent is connected to a repo, ask it to commit and push.
• Share a link — for hosted content (like HTML pages), your agent can provide the URL.
