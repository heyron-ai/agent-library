Dashboard Blank Screen? (No Error Message)

A fast recovery flow for the "just white/black screen" issue when your agent page won’t render.

If your Heyron dashboard opens to a blank screen (with no useful error), this is usually a browser/session problem first — not account loss.

Most blank-screen cases resolve in under 5 minutes with a clean session reset + extension/network isolation.

Fast fix checklist (run in order)

1) Hard refresh first

Mac: Cmd+Shift+R · Windows: Ctrl+Shift+R. If mobile, fully force-close the app/browser and reopen.

2) Open the same page in Incognito/Private mode

If it works there, your normal profile has stale cache/cookies or an extension conflict.

3) Disable ad/privacy extensions for Heyron

uBlock/privacy tools can block required scripts or websocket traffic and produce a blank render.

4) Turn off VPN/proxy temporarily

Some VPN routes and strict proxies break script/API responses. Quick no-VPN retry helps isolate this fast.

5) Try a second browser and second device

This tells you immediately whether the issue is local (browser/device) or account/platform-wide.

Quick isolation matrix

  • Works in incognito, fails in normal browser: stale session cache or extension conflict
  • Fails on one browser only: browser-specific cache/profile issue
  • Fails on all browsers on same network: VPN/proxy/DNS filtering likely
  • Fails everywhere + Discord also quiet: agent/service-side issue (escalate)

Advanced checks (2 more minutes)

6) Clear site data for heyron.ai only

Clear cookies/cache for the Heyron domain only, then sign in again. This avoids nuking your whole browser history.

7) Check DevTools console once (optional)

If you see repeated blocked, CSP, ERR_*, or websocket failures, include one screenshot in support.

8) Compare Wi-Fi vs mobile hotspot

If hotspot works, your home/work network path is likely filtering or timing out required requests.

When to escalate to #help

  • Blank screen persists longer than 10 minutes
  • You tested incognito + second browser/device and still fail
  • Discord/Telegram is also non-responsive for the same agent

Copy/paste support packet

My Heyron dashboard is loading as a blank screen. Started around: [time + timezone] Agent URL/subdomain: [your link] What I tried: hard refresh, incognito, disabled extensions, VPN off, second browser/device Result: [still blank / intermittent / works on hotspot only] Signup email: [your account email]

Security reminder: never post tokens, API keys, or passwords in #help.

Bottom line: blank screens are usually local rendering/session issues. Use the isolation flow above to identify the failing layer quickly and escalate with clean diagnostics if needed.
