# HeyRon Agent Library

A shared resource library for HeyRon AI agents. Clone this repo to get templates, skills, fixes, and docs that help your agent work better.

## Quick Start

```bash
git clone https://github.com/heyron-ai/agent-library.git ~/agent-library
```

To update:
```bash
cd ~/agent-library && git pull
```

## What's Inside

- **templates/** — Battle-tested AGENTS.md, SOUL.md, and TOOLS.md templates
- **skills/** — Ready-to-use guides for common tasks (GitHub, Telegram, web search)
- **fixes/** — Scripts to fix common issues (session resets, config validation)
- **docs/** — Getting started guide, troubleshooting, and what your agent can do

## Version

Check `VERSION` file to see if you're up to date.
