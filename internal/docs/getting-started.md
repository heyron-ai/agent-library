# Getting Started with Your HeyRon Agent

Welcome! Your AI agent lives in its own container and is ready to help you. Here's what you need to know.

## Talking to Your Agent

- Visit your agent's URL (the one you got after signing up)
- Click **Connect**
- Start chatting!

## What Your Agent Can Do Right Now

- **Chat & brainstorm** — Ask anything, bounce ideas, get advice
- **Write code** — Python, JavaScript, HTML, bash, and more
- **Create files** — Documents, scripts, websites, data files
- **Search the web** — Research topics, find information
- **Run commands** — Your agent has a full Linux terminal
- **Remember things** — Ask it to write notes to files for persistence

## What Needs Setup First

These require you to provide credentials:

| Feature | What You Need | Guide |
|---------|--------------|-------|
| GitHub | Personal access token | [GitHub Setup](../skills/github-setup/README.md) |
| Telegram | Bot token from @BotFather | [Telegram Setup](../skills/telegram-setup/README.md) |

## Tips for Best Results

1. **Be specific** — "Build me a landing page for a dog walking business" works better than "make a website"
2. **Ask for files** — Your agent can create real files you can download
3. **Start fresh if stuck** — Refresh your browser and reconnect for a clean session
4. **Check the file browser** — Click the files icon to see what your agent has created

## Getting Help

- Post in **#help** on the [HeyRon Discord](https://discord.gg/saAPheKSfB)
- Describe what's happening and your agent's name
- The team will help!
